# تطويرات (سجل طلبات / خارطة طريق)

استخدم هذا المجلد لتثبيت **ما تريد بناؤه لاحقًا** دون خلطه بكود الإنتاج.

## مقترح تسمية الملفات

- `YYYY-MM-DD-اسم-الخاصية.md` (مثال: `2026-04-30-تصدير-Excel.md`)

## محتوى مقترح لكل ملف

1. **المشكلة / الفائدة** للمستخدم النهائي  
2. **نطاق العمل:** صفحات، جداول DB، صلاحيات  
3. **الارتباطات:** روابط لملفات في `src/` أو `prisma/` ستُلمَس  
4. **معايير القبول** (اختبارات يدوية أو آلية)  
5. **مخاطر أمنية/أداء** إن وُجدت  

بعد التنفيذ الفعلي في الكود، يُفضَّل الإشارة من هنا إلى الـ PR أو الـ commit.
