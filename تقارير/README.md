# تقارير المشروع (WMS)

هذا الفرع مخصص **للتوثيق المرجعي** وليس لتشغيل التطبيق.

| المسار | الغرض |
|--------|--------|
| `تقارير/الخصائص/الهيكلية-الصحيحة-والمرجعية/` | خريطة الارتباطات، دليل إضافة خصائص، أمن ونشر |
| `تقارير/الخصائص/تطويرات/` | أفكار وخصائص مخططة (سجل طلبات) |
| `تقارير/الخصائص/مخالفات-الممارسات-وتحسينات/` | ديون تقنية، تكرار، مخالفات ممارسات |

**جذر التطبيق البرمجي:** مجلد `wms` (حيث `package.json` و`src/`).

**لا تُخزَّن أسرار** داخل هذه الملفات (لا نسخ `.env`، لا كلمات مرور، لا مفاتيح JWT).
